# 🏌️ Golf Swing Check App

ゴルフスイング撮影・解析のための個人開発プロジェクト。

MVPでは  
- 📱 iPhone：撮影専用アプリ  
- 📲 iPad：受信・表示専用アプリ  

として構成する。

---

# 🎯 プロジェクト目的

- 高品質なスイング動画を撮影
- 別端末でリアルタイム表示
- 将来的にスイング解析機能を追加
- 最終的には弾道測定・AI解析まで拡張

---

# 🧱 アプリ構成

## ① 撮影アプリ（iPhone）

役割：
- カメラ制御
- 高FPS撮影
- 動画データ送信

### MVP機能
- カメラ起動
- 録画開始 / 停止
- ローカル保存
- iPadへ動画送信

---

## ② 受信アプリ（iPad）

役割：
- 動画受信
- 動画再生
- フレーム単位確認

### MVP機能
- 受信待機
- 動画保存
- 再生
- スロー再生

---

# 🔌 通信設計（MVP）

- 同一Wi-Fi内通信
- P2P通信想定
- 低遅延優先
- セキュリティは後回し（MVP）
